Auto Delivery Task with Sales Order

This module allows creating delivery status automatically and allows to track its status through project task. Ideal for ecommerce scenario where delivery is managed internally or with third party.

Dependencies
  • sale.
  • project.
Backend Configuration

In configuration, set the name of the project and starting and end stage.

Quick View
  • On creating and confirming Sales Order, a delivery task is auto created.
  • This delivery task can be tracked as per its delivery status.
  • Upon completion of delivery, status is changed to done in Delivery Order automatically.
